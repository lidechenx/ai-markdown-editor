# 参与贡献

感谢你愿意改进本项目。

## 本地开发

```bash
npm install
npm run dev
```

## 提交前检查

```bash
npm run test
npm run build
```

## 提交说明

请用完整句子描述「改了什么、为什么」，便于审阅与生成变更日志。

若有与**使用者可见行为**相关的改动，请同步更新根目录 **`README.md`**（README 仅面向终端用户：安装、使用、隐私、限制；勿写入维护者内部或运营类内容）。

技术细节、给协作者的深度说明可写入 `docs/` 下**会提交**的普通 `.md`；维护者私有笔记请使用 `docs/*.local.md`（已被 `.gitignore` 排除）。
